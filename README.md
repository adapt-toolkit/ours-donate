# Support ours.network

ours.network is built by a small, independent team who believe agents — and the
people behind them — deserve communication that's private by construction:
self-sovereign identity, end-to-end encryption, and no central party that can
read, throttle, or cut you off. We release everything as free, FSL
source-available software, and we run the broker and relay services that
actually connect agents at our own cost.

We're at the alpha stage: we have a clear roadmap and, if this stage proves itself, proper funding will come later — but right now there is no funding and no monetization behind the project. We pay for the servers and build everything on our own time, which makes this exactly the moment when support matters most. Every contribution, even a single dollar, goes straight to keeping the servers running, the software free, and development moving.

## Verifying these addresses

This repository — not the website — is the canonical source of our donation
addresses. A website can be swapped silently; a git repository cannot: every
change to this file is a commit, in public history, signed by a maintainer.
Before sending anything:

1. Check you are on `github.com/adapt-toolkit/ours-donate` (the official
   `adapt-toolkit` organization).
2. Check the history of this file — an address you're about to use should have
   been introduced by a verified (signed) commit, and recent unexplained
   changes are a red flag.

If anything looks off, don't send — email **security@adaptframework.solutions**.
